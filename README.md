Cómo escribir CSS
====

* Poner todos los atributos para cada selector en orden alfábetico
* Mantener selectores de cada sección en orden de aparición
* Para poner nombre: 
	* 'seccion'-'identificador', por ejemplo 'project-gallery'
	* no usar mayúsculas
	* usar - en vez de _
